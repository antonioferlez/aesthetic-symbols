# Aesthetic Symbols

A web gallery of aesthetic symbols and emojis that allows users to search, copy them to the clipboard, and mark them as favorites.

## Features
- Search by name or tags
- Copy symbols with a single click
- Favorites system using LocalStorage
- Automatic sorting by most liked symbols
- Responsive and minimalist design

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
